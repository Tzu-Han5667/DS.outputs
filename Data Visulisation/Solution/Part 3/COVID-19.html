Case Amount

By case amount, we can realise monthly cases in each country. Also, we can figure out growth of COVID-19.

Death Rate

By groupby command we can get monthly deaths and confirmed case, moreover, by function deaths/confirmed, we can get monthly death rate. Death rate could demonstrate the most difficult period of COVID-19 in each country. And tell which month is the worst time.

World Route

After combining 2 original csv files, we can find the earliest date of cases in each country. By these dates, we can know the spread path of COVID-19.
